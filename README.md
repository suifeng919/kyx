# MoneyTracker 个人记账本

**学生个人消费记账工具** — 轻量、快速、本地化

## 项目结构

```
MoneyTracker/
├── MoneyTracker.WinForm/    # UI层：WinForm窗体
├── MoneyTracker.Core/       # 业务层：模型+服务
├── MoneyTracker.Data/       # 数据层：SQLite+Dapper
└── MoneyTracker.Tests/      # 单元测试
```

## 技术栈

C# .NET 8.0 + WinForm + SQLite + Dapper + EPPlus
